NeuroGolf (ARC-AGI) ↗ — compiling reasoning tasks into the smallest possible neural circuits把推理任务编译成尽可能小的神经电路 research
  • The insight that unlocked it: this isn't a learning problem — each task's true rule is shipped as a generator, so the job is to compile known rules into minimal networks, not to train.打开局面的洞察：这根本不是学习问题 —— 每道题的真规则以生成器的形式给出，所以要做的是把已知规则编译成最小网络，而不是训练。
  • Reverse-engineered the scoring into a cost model, then built the tooling to construct and validate circuits against freshly generated data so nothing overfits the public samples.把评分逆向成一个成本模型，再做出一套工具来构造电路、并在现场生成的数据上验证，确保不会过拟合公开样本。
  • What I took from it: the difference between fitting something that passes the samples and implementing the rule that produces them — only the second holds up. Written up as a reusable methodology note.我的收获：分清"拟合一个能通过样本的函数"和"实现产生这些样本的规则" —— 只有后者站得住。写成了一份可复用的方法论总结。
